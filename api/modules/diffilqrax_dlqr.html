diffilqrax.diff_lqr.rev_dlqr(dims, res, tau_bar)[source]#

Reverse mode for DLQR. Find the reverse gradient by solving LQR problem on the optimal trajectory and the gradients wrt to tau_star.

Notes

\(\overline{q}\), \(\overline{r}\), \(\overline{a}\) from solving the reverse LQR problem where \(q_{\text{rev}} = \overline{x}\), \(r_{\text{rev}} = \overline{u}\), \(a_{\text{rev}} = \overline{\lambda}\) which is set to 0.

Use build_ajoint_lqr() to build the reverse LQR problem with gradients. Where \(\overline{c} = [\overline{q}, \overline{r}]\), \(\overline{F}\), where \(F = [A, B]\), and \(\overline{C}\) (where \(C = [Q, R]\)) as,

\[\overline{C} = 0.5 \left( \overline{c} \tau_{\star}^{T} + \tau_{\star} \overline{c}^T \right)\]
\[\overline{F}_t = \lambda_{\star_{t+1}} \overline{c}^T + f_{t+1} \tau_{\star_t}^T\]
Parameters:

  • dims (ModelDims) – The dimensions of the model.

  • res (Tuple[LQRParams, Tuple[Array, Array, Array]]) – The result from the forward pass, that is, the tuple of LQR gradients and optimal tau gradients.

  • tau_bar (Array) – The gradients with respect to tau.

Returns:

The LQR parameters with reverse gradients.

Return type:

LQRParams
