Associative LQR#

LQR solver using associative parallel scan

diffilqrax.plqr.associative_opt_traj_scan(model, etas, Js, alpha=1.0)[source]#

Obtain effective state dynamic and bias through associative scan. Ks returned (Kx, Kv, Kc, delta) to recover optimal control: state feedback gain, linear val fn gain, dynamics offset gain, delta of linear gains.

Parameters:

  • model (LQRParams) – LQR problem (initial state, LQR parameters).

  • etas (Array) – Linear value functions.

  • Js (Array) – Quadratic value functions.

  • alpha (float, optional) – Linesearch step parameter, by default 1.0.

Returns:

Effective state dynamics, effective bias, Ks, offset.

Return type:

Tuple[Array, Array, Tuple[Array, Array, Array, Array], Array]
